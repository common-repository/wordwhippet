== Description ==

WordWhippet is a free optmisation plugin with multiple options to speed up your WordPress site & improve your site's search engine rankings & PageSpeed Insights scores.

Options included:

* Minify site source code
* Enable Cache Control Headers
* Async Javascript
* Defer Javascript
* Enable Gzip Compression
* Remove Query Strings
* Disable hotlinking
* Disable Emoticons
* Disable Comment Scripts
* Disable Auto Embedding
* Disable Jquery Migrate
* Disable Really Simple Discovery
* Disable Shortlinks
* Disable XML_RPC
* Remove WLManifest Link
* Disable self-pingback
* Disable heartbeat api
* Disable Dashicons on the front-end
* Disable or limit post revisions
* Hide WordPress/WooCommerce Version tags

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install it via the WordPress Plugin Repository
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Toggle optimisation options on/off on the WordWhippet settings page and save your settings
